Scenario Setup

Little Lemon restaurant requires an update to their booking systems. You can use your knowledge of databases to help them carry out these updates. There are several aspects that need to be considered when engaging with this project including:

  • Maintaining consistency when updating code.

  • Ensuring there are sufficient structures in place to save, alter and query data.

  • Ensuring there is a mechanism for displaying the data in a manner that is easily interpretable.

  • Checking that there is a front-end system for accessing and updating the data.

Working Environment

GIT

Writing an application never happens in a single day. It takes time and many revisions before one completes the application to an acceptable standard. To ensure consistency throughout the writing process it is beneficial to employ some form of version control software.

In this course you have had experience with using Git, a version control system. In addition to tracking the timescale of various updates, it also enables many different programmers contribute the production at the same time. Git can be downloaded from this location https://git-scm.com/downloads . For more information on its installation and use, please consult the subsequent readings.

MySQL

To access and store information for the Little Lemon restaurant it is necessary to create a database. There is a plethora of different database types that can be used, such as: Oracle, SQLite, PostgreSQL and MongoDB to name a few.

In previous courses you have had experience with MySQL. MySQL is a unified visual tool for database creation and management. It comes with several additional applications that enable smooth interaction with the data. Data can be accessed using MySQL workbench,directly using the command line, through an application client such as C/Python or Java as well as using the MySQL command-line client.

More information is provided in the subsequent lectures. You can also find, download and implementation instruction here https://dev.mysql.com/downloads/

Python

There are many different programming languages that take in user input and store it in the database. MySQL connectors are a suite of drivers that enable interaction between MySQL and several languages including: .net, Node.js, Python, C++ and C.

There are always strengths and weaknesses associated with any given task. Some people may find that they have an affinity for different programming languages depending on their own preference and experience.

Python is a high-level general-purpose language that was built with readability in mind. It is dynamically typed and has many built-in features that simplify the process of coding with it. In addition, it is supported by Oracle through the driver mysql-connector-python for interacting with a MySQL database.

In this course Python is the preferred method for communicating with the database and updating the data based on user input. The subsequent lectures cover the implementation and setup of python. To download Python please go to https://www.python.org/downloads/ for an installation that is appropriate to your machine.

Tableau

Business intelligence is gained from querying existing information and identifying existing patterns in the data. Over the years, businesses have been gathering customer data and storing it in so-called data lakes. There has been a huge focus in recent times on analyzing this data and using it to make business decisions that can result in more profitable returns.

To better analyze data, one needs to be able to extract subsets, generate meaningful statistics and identify common trends. This task is difficult to achieve when the data exists as rows and columns in a database or excel sheet. Therefore, there have been many applications developed to aid the programmer in this task. These include PowerBI, Tableau and Qlik.

In this course Tableau is showcased as a means of extracting pertinent information and displaying it for analysis. For more information pertaining to Tableau please consult https://www.tableau.com/ as well as the subsequent lectures on visualizing information through tableau.
