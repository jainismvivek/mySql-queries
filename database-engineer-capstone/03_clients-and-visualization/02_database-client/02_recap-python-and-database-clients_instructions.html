Working Environment

A drive is required to facilitate communication between Python and MySQL. In preceding courses, you have covered the installation and workings of mysql-connector. MySQL connector is a driver for Python and MySQL which takes Python syntax and passes it to MySQL in a fashion that allows for the running of MySQL queries.

This driver can be installed on the command line or directly from the Jupyter notebook using a pip install command. The connector/python is a driver for interacting with MySQL through a range of programming languages including C, C++, Node.js to name a few.

MySQL Connector/Python offers support for all features that are capable within MySQL. This includes standard queries, transactions and procedures. It provides full conversion capabilities between Python and MySQL datatypes. Any required conversion happens behind the scenes. For example, the date time function in MySQL behaves differently to MySQL date time. However, a lot of this heavy lifting is done automatically without requiring intervention from the programmer. In addition, MySQL Connector/Python provides data compression for faster communication between client and server.
